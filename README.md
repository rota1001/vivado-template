# Vivado Template
Just a template to develop hardware on Kintex-7 FPGA with Vivado CLI tools.

## How to use
Clone:
```
git clone https://github.com/rota1001/vivado-template.git
```
Build:
```
make build
```
Flash:
```
make flash
```
Simulate:
```
make sim
```
And you can see the wave.vcd in the build/ directory.
